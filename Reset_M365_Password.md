# How to Reset a Microsoft 365 Password

## Overview
This guide provides step-by-step instructions to reset a Microsoft 365 password for end-users. It is intended for technical support teams to assist employees or clients efficiently.

## Steps

1. **Sign in to Microsoft 365 Admin Center**
   - Go to [https://admin.microsoft.com](https://admin.microsoft.com)
   - Use an admin account to access user management.

2. **Locate the User**
   - Navigate to **Users â†’ Active users**.
   - Search for the account needing a password reset.

3. **Reset Password**
   - Select the user, then click **Reset password**.
   - Choose **Auto-generate password** or set a **Custom password**.
   - Ensure **Require this user to change their password when they first sign in** is checked.

4. **Notify the User**
   - Send the new password securely via approved communication channels.
   - Confirm that the user can successfully log in.

## Troubleshooting
- If the user cannot log in after reset, verify:
  - Account is **active** and **not blocked**.
  - Correct **username/email** is being used.
  - Multi-factor authentication (MFA) settings do not prevent access.
